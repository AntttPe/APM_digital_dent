# APM Dental Lab — apmdental.pl

Strona internetowa cyfrowego laboratorium dentystycznego APM Dental Lab. Skierowana do stomatologów, ortodontów i właścicieli klinik zamawiających szyny stomatologiczne drukowane 3D.

## Produkty

- **Szyny retencyjne** — `/produkty/szyny-retencyjne`
- **Szyny relaksacyjne** — `/produkty/szyny-relaksacyjne`
- **Szyny do wybielania** — `/produkty/szyny-do-wybielania`

## Stack

- Next.js 16 (App Router) · React 19 · TypeScript
- Tailwind CSS v4 · Framer Motion 12
- Wdrożenie: Vercel

## Rozwój lokalny

```bash
npm install
npm run dev
```

Strona dostępna pod `http://localhost:3000`.
